
# 📱 SmartphoneApp

A modern and stylish iOS app built with SwiftUI to showcase and shop for smartphones across brands and categories. This app simulates an e-commerce mobile experience with product browsing, category filters, cart and checkout functionality.

## 🛠️ Features

- Browse smartphone categories (flagship, budget, etc.)
- View product details and high-res images
- Add/remove products from cart
- Responsive checkout form
- Order confirmation dialog
- Clean, modern UI with SwiftUI

---

## 🧰 Tech Stack

- SwiftUI
- MVVM Architecture
- Xcode Simulator (iPhone 16 Pro)
